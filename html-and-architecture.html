Page Titles and Meta Descriptions

Titles and Meta descriptions are a fundamental element of on page optimisation. They are a good opportunity to tell both search engines and users what information they will see on the page.

Meta Title

Page titles are also visible to users in the search results, and provide an opportunity to boost click through rate.

The meta title sits in the <head> of a page’s HTML and appears in search results as the link to the page:

When looking at page titles, there are several different factors which we need to take into consideration:

  • Ensure that each page has a unique title
  • The title should be no longer than 512 pixels wide (for desktop), otherwise it will be truncated or rewritten
  • This equates to around 40-55 characters, depending on which letters are used
  • It should feature the page’s target keyword first
  • If there is space, add a variant or co-occurring term in the middle of the title
  • Include the brand at the end of the title
  • Separate sections with hyphens (-) or pipes ( | )
  • An example optimized title is: Mobile Broadband & Internet Dongles | Three

Meta Description

Meta descriptions sit in the <head> of a page’s HTML and are the 928 pixel width visible piece of text below the page titles in search results.

Different to page titles, they are no longer used as a ranking signal. Copy in the meta descriptions should be used to boost click through rate, but remain accurate as to reflect the content on the page, otherwise search engines may deem it too irrelevant and display text generated from the webpage instead.

  1. Ensure that each page has a unique description
  2. Descriptions don’t count towards Google’s ranking algorithm, but do directly impact on the likelihood of users clicking through to the page from search results
  3. Google bolds search queries, synonyms and co-occurring keywords when they appear in descriptions, making it prudent to optimize them
  4. The maximum description width is approximately 920 pixels (for desktop), beyond which it will be truncated or rewritten
  5. This equates to around 120-145 keywords, depending on which letters are used
  6. An example optimized description is: Looking for a dongle, tablet or mobile Wi-Fi connection? Join the UK's fastest growing network and get online wherever you are.
